# Chat_me

Chat_me is a personalized Retrieval-Augmented Generation (RAG) chatbot that
suggests replies in my writing style based on semantic similarity with my past conversations.

## Features
- Sentence embeddings using Sentence Transformers
- Semantic search with FAISS
- Memory-based reply suggestions
- Clean, modular Python design

## Tech Stack
- Python
- Sentence-Transformers
- FAISS (CPU)
- JSON-based memory
